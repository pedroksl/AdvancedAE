# AdvancedAE More Upgrades

A fork of [AdvancedAE](https://github.com/pedroksl/AdvancedAE) by [Pedroksl](https://github.com/pedroksl). Available on [https://github.com/Nobody1902/AdvancedAE](https://github.com/Nobody1902/AdvancedAE).

Originally from [Extended AE](https://www.curseforge.com/minecraft/mc-mods/extendedae), AdvancedAE focuses on convenience and improving user experience when handling your ME System.

## Added features:
- ### Auto Fluid Upgrade
  - Automatically tops off partially filled fluid-containing items in the player's inventory using fluids available from the linked ME network.
  - It will not fill empty containers to avoid introducing incompatible fluids. Tested with Mekanism (gases not supported) and Ad Astra.

## Quantum Armor

AdvancedAE adds Quantum Armor - A powerful armor that can store a massive amount of items and fluids, with support for crafting directly from the armor.

## Advanced Pattern Providers

AdvancedAE adds Advanced Patterm Providers. They are crafted using ExtendedAE's pattern providers, and retain the improved inventory size but with additional features!

By utilizing an Advanced Pattern Encoder, you can modify Processing Patterns to select to which face each input will be sent, eliminating the need for multiple pipes and used sides to interact with machines that require them.
